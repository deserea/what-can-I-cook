## What it does
- Accepts a list of ingredients from the user
- Loads recipe data from a JSON file
- Compares ingredients against a set of recipes
- Suggests the best matching meal ideas
- Shows which ingredients you already have
- Shows which ingredients are missing
- Builds a shopping list from missing ingredients
- Allows user to select a recipe and generate a focused shopping list
