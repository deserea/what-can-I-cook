# What Can I Cook?

A Streamlit cooking assistant for American home kitchens.

## Features

- Enter ingredients you definitely have
- Add ingredients you may also have
- Add ingredients you are willing to shop for
- Filter by cuisine
- Filter by meal type
- Choose cooking level:
  - Beginner
  - Intermediate
  - Advanced
- Choose time commitment:
  - Quick meals
  - 1 hour or less
  - 2 hours or less
  - More advanced / longer cook
- Choose how many people the meal should feed
- View recipe cards
- View detailed instructions
- Get shopping suggestions
- Get chef-level upgrade ideas

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
